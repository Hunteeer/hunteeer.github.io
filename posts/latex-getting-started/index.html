安装 MacTeX

MacTeX 是 TeX Live 的分支，专门支持 macOS。TeX Live 又是什么呢？简单来说，TeX Live 是一个 TeX 排版系统的套装（全家桶），包含排版引擎、宏包、管理工具、编辑器、字体等。TeX Live 保持每年更新的频率，版本号是年份（比如 TeX Live 2018）。通过 TeX Live 自带的管理工具，可以对宏包、字体等进行更新，但像 TeX、LaTeX 等命令行工具是随着版本号更新的。

安装 MacTeX 有两种方式，一是下载 MacTeX.pkg 进行完全安装。这种方式会安装所有的内容，包括 TeX Live 提供的所有宏包，好处是能避免很多麻烦。但 MacTeX.pkg 文件非常大，有好几个 GB，而且绝大多数宏包你可能这辈子都不会用到。如果你刚刚接触，还是推荐这种方式。另一种方式是下载 BasicTeX.pkg 进行最小安装。这种方式只会安装必要的命令行工具，以及一部分重要的宏包和字体。编辑器等 GUI 工具也不会被安装。但是之后可以自行安装。我采用第二种方式。

下载 PKG 包，安装即可。也可以通过 HomeBrew 安装：

$ brew cask install basictex

安装安成后，从命令行输入以下内容来验证：

$ tex '\empty Hello world!\bye'
This is TeX, Version 3.14159265 (TeX Live 2018) (preloaded format=tex)
[1]
Output written on texput.dvi (1 page, 224 bytes).
Transcript written on texput.log.

MacTeX 的卸载：可参考官方文档 Uninstalling TeX。

宏包管理

MacTeX 自带了 GUI 版本的管理工具 TeX Live Utility，以及命令行版本的管理工具 tlmgr。前面说过了，BasicTeX 不会安装 GUI 的工具，因此我们使用 tlmgr 来管理。

列出宏包目录：

$ tlmgr list  # 列出所有宏包。前面有 i 的表示已经安装
$ trmgr list --only-installed  # 列出已经安装的宏包

查找宏包信息：

$ tlmgr info ctex  # 查找 CTeX 信息
package:     ctex
category:    Package
shortdesc:   LaTeX classes and packages for Chinese typesetting
longdesc:    ctex is a collection of macro packages and document classes for LaTeX Chinese typesetting.
installed:   Yes
revision:    47595
sizes:       doc: 1141k, run: 717k
relocatable: No
cat-version: 2.4.14
cat-date:    2018-05-02 16:00:12 +0200
cat-license: lppl1.3c
cat-topics:  chinese book-pub class
collection:  collection-langchinese

安装 & 卸载宏包：

$ tlmgr install ctex  # 安装 CTeX
$ tlmgr install --with-doc ctex  # 安装 CTeX 及其文档
$ tlmgr uninstall <package>  # 卸载 package

更新宏包：

$ tlmgr update --list  # 列出有更新的宏包
$ tlmgr update --all  # 更新所有宏包

如果习惯了使用 xPM 之类的工具，BasicTeX 是更好的选择。tlmgr 有个缺点是不能自动解决依赖关系。不过也不是没有办法，宏包 texliveonfly 可以解决这个问题。它能在编译文档的时候，自动安装依赖的宏包。

编辑环境

macOS 上比较有名的 TeX 编辑器有 TexPad、TeXstudio、TeXShop、TeXworks、TeXMaker。还可以使用 VIM、Emacs、Sublime Text、Atom、VScode 这类编辑器，通过安装插件来支持。我喜欢 Atom。

Atom 安装以下相关插件后，就能获得一个很舒服的 LaTeX 编辑器了：

  • language-latex：语法高亮
  • latex：编译
  • autocomplete-latex：自动补全
  • autocomplete-latex-cite：自动补全文献引用
  • autocomplete-latex-references：自动补全交叉引用
  • document-outline：文档大纲

其他

texdoc

使用 texdoc 查看宏包文档，就像使用 man 一样方便。前提是安装宏包的时候要携带安装 DOC。宏包文档一般是 PDF 格式，放置在 ${TEXLIVE}/texmf-dist/doc 目录下。直接输入 texdoc <package>，texdoc 会使用默认的 PDF 阅读器直接打开相应的 PDF 文档。如果想进行模糊搜索可以使用 -s 参数：

$ texdoc -s ctex
 1 /usr/local/texlive/2018basic/texmf-dist/doc/latex/ctex/ctex.pdf
   = [zh] Package documentation
 2 /usr/local/texlive/2018basic/texmf-dist/doc/latex/ctex/README.md
   = Readme
Enter number of file to view, RET to view 1, anything else to skip:

texhash

有时，你可能会手动安装一个文件到 TeX Live 的文件树中，但是 TeX Live 并不能找到，需要使用以下命令刷新索引：

$ texhash

比如安装符合国家标准的文献著录样式：GBT7714-2005-BibTeX-Style

LaTeXiT

这是一个 GUI 工具，完整版的 MacTeX 是默认安装的。利用它可以将 LaTeX 公式转换为 pdf、png、eps 等格式，非常方便。我觉得可以手动安装一个。

pdffonts

不知道你有没有遇到过这种情况，PDF 文件在自己电脑上好好的，到别人电脑上查看就乱码了。这是因为使用了无法嵌入的字体（或者压根没有嵌入字体），这种字体往往是有版权的，当然，能嵌入也不一定代表没版权。pdffonts 可以快速的让你知道 PDF 文件中的各种字体的嵌入情况。例如

$ pdffonts index.pdf
name                                       type              encoding         emb sub uni object ID
------------------------------------------ ----------------- ---------------- --- --- --- ---------
FTNAMI+TimesNewRomanPSMT                   CID TrueType      Identity-H       yes yes yes      6  0
UMRIQY+SourceHanSerifCN-Regular-Identity-H CID Type 0C       Identity-H       yes yes yes      8  0
