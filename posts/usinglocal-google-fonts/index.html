使用本地 Google Fonts

2017-03-05
折腾博客
949
    • 使用 Google Chrome Console
    • 本地化 Google Fonts

我的博客国内访问速度特别慢，今天仔细找了下原因，发现原来是主题使用了 Google Fonts。这已经是一个很普遍的老问题了，作为新人，我被这个小问题磨了好久啊，一度怀疑是不是服务器的问题。最终我将使用到的 Google Fonts 下载到服务器上从而解决了这个问题。在发现、分析、解决问题的过程中，接触到很多新的东西，所以我觉得有必要记录下来。

使用 Google Chrome Console

对于博客访问速度慢的问题，我真的是束手无措，因为完全不清楚问题出现在哪个环节。Chrome Console 是一个很好用的工具，当我使用它测试我的博客之后，真的感觉是打开新世界的大门，有了解决问题的方向，之后的每一步都有了根据。

Console 可以调试很多东西，本次用到的是 Network。打开它，再打开博客，各个文件的请求、状态、时间等一目了然。发现

http://fonts.googleapis.com/css?family=Lato:300,300italic,400,400italic,700,700italic&subset=latin,latin-ext

这个请求耗了很长时间，最后还加载失败了。很明显是被**了。修改 NexT 配置文件，禁用字体，再次测试博客加载速度，从来没体验过的快感😭。

本地化 Google Fonts

对于这个普遍的问题，有很多解决方法：

  • 禁用
  • 换字体
  • 换国内源，比如使用 360 CDN Google 字体库、科大 CDN Google 字体库。貌似都挂了。
  • 使用本地 Google Fonts

老实说，Google 的东西我还是很乐意用的，将Google Fonts 放在自己服务器应该是最好的方法了。

首先明确需要使用那些字体，NexT 主题的字体配置在博客根目录下 themes/next/layout/_partials/head/external-fonts.swig。使用了 Lato:300,300italic,400,400italic,700,700italic。

前往 Google Fonts 下载相应的字体，在博客根目录的 source 目录新建一个 fonts 目录用来存放字体文件，将下载好的字体解压并放到里面，还需要在里面新建一个 fonts.css 文件。最后目录结构如下

source/fonts
 - fonts.css
 - Lato		#存放字体文件

按照 CSS 的语法规则编写 fonts.css：

@font-face {
      font-family: <YourWebFontName>;
      src: <source> [<format>][,<source> [<format>]]*;
      [font-weight: <weight>];
      [font-style: <style>];
    }

Lato: 300 的 @font-face 如下

@font-face {
  font-family: 'Lato';
  font-weight: 300;
  font-style: normal;
  src: url('Lato/Lato-300/Lato-300.eot');
  src: url('Lato/Lato-300/Lato-300.eot?#iefix') format('embedded-opentype'),
       local('Lato Light'),
       local('Lato-300'),
       url('Lato/Lato-300/Lato-300.woff2') format('woff2'),
       url('Lato/Lato-300/Lato-300.woff') format('woff'),
       url('Lato/Lato-300/Lato-300.ttf') format('truetype'),
       url('Lato/Lato-300/Lato-300.svg#Lato') format('svg');
}

把其他字体的 @font-face 也写进 fonts.css。

修改 NexT 字体配置。在 NexT 主题配置文件 _config.yml 里启用 font，并将其 host 设置成存放 fonts.css 的目录 /fonts，因为经过 Hexo 渲染并部署到服务器上后，fonts 目录处于网站根目录。

修改 external-fonts.swig，做个小改动：

Before:

<link href="{{ font_host }}/css?family={{ font_families }}" rel="stylesheet" type="text/css">

After:

<link href="{{ font_host }}/fonts.css?family={{ font_families }}" rel="stylesheet" type="text/css">

最后测试博客的加载速度，

Request URL:http://.../fonts/fonts.css?family=Lato:300,300italic,400,400italic,700,700italic&subset=latin,latin-ext
Request Method:GET
Status Code:200 OK

成功加载。
