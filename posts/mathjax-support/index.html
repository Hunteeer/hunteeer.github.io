Mathjax：让博客支持 LaTeX 公式

MathJax 是一个开源的 JavaScript 公式渲染引擎，支持 LaTeX, MathML, AsciMath 语法。支持所有现代化的浏览器，包括移动设备上的。将 MathJax 引入博客写作非常简单，配合 Markdown，写涉及到公式的文章更加简洁高效。

Why MathJax？

接下来的一段时间可能需要写一些公式，一件让人头大的事情。搜索了下关于 Markdown 里面插入公式的方法，然后找到了 MathJax。它吸引我的地方在于可以直接渲染出公式，而不是类似于图片的形式，我完全不用担心公式的后续问题，只要有代码在，公式就在。当然，真正的问题是 LaTeX、MathML 等公式语法的学习。关于这一部分可以参考我的下一篇博文：使用 TeX 编写公式：符号和用法总结。本文只是就如何引入 MathJax 问题的简单记录，更详细的内容参考MathJax 官方文档。

Getting start

将以下代码写到文章源文件中，那么该文章中的符合语法的公式就能被渲染出来了。

<script type="text/javascript" async
  src="https://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-MML-AM_CHTML">
</script>

以下是官方给出的Example：

<script type="text/javascript" async
  src="https://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS_CHTML">
</script>
When \\(a \ne 0\\), there are two solutions to \\(ax^2 + bx + c = 0\\) and they are
$$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$

以下是渲染之后的样子，很不错吧。

When \(a \ne 0\), there are two solutions to \(ax^2 + bx + c = 0\) and they are $$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$

这种方法使用 MathJax 官方提供的 CDN（Content Distribution Network）服务，不需要自己在本地或服务器上搭建 MathJax 引擎，几行 JavaScript 就够了。自己搭建也是可以的，参考：

  • Installing Your Own Copy of MathJax - MathJax documentation
  • mathjax/MathJax - GitHub

LaTeX support

接下来插入公式代码。我对比之后决定用LaTeX。公式和文章其他部分是可以混排在一起的，因此，首先就要告诉 MathJax 公式代码的位置。MathJax 支持两种形式的公式插入：inline（行内）/displayed（行间）。默认的标记方法如下：

Type Mark
inline \(...\)
displayed $$...$$ OR \[...\]

需要注意的是，\ 默认是转义符，需要使用 \\。比如应当使用 \\(...\\) 来插入行内公式。

两种方式的区别可以查看上文官方Example。

然后就是 LaTeX 语法的学习。也有许多可视化甚至手写识别转化 LaTeX 代码的工具，可自行搜索。

Hexo support

本博客使用 Hexo 搭建，这部分内容关于如何简化写公式的步骤。实际上 Hexo 有一个专门的插件 hexo-math 支持 MathJax，参考：

  • akfish/hexo-math - GitHub

不过最后我还是决定用 Hexo 的模板功能来解决。麻烦的地方无非在于每次都需要插入 JavaScript，模板功能完全可以解决。在 scaffolds 目录下建立一个 math.md 模板，可以直接复制一份 post.md 的模板文件，改个名字，插入 MathJax 的 JavaScript，以后需要写公式的时候就依照这个模板新建文章：

$ hexo new math <post_name>
