更改 Android 字体

本篇记录手动更改 Android 字体的过程，简单几步即可实现，体会 Android 的乐趣。

前提

前面某篇文章我提到了本博客的字体问题，这几天想更改下手机的字体。我的目标是将英文非衬线字体改成 Lato，中文字体改成思源黑体，均为开源字体，可以在相关网站下载到。开始之前需要以下条件：

  • ROOT required
  • 字体文件：Lato、思源黑体
  • 字体编辑器

过程

Android 字体储存在 /system/fonts。字体 XML 文件是 /etc/fonts.xml。替换字体的方法简单地说就是把新字体放置到 /system/fonts 下，然后修改 XML 文件，把对旧字体的描述代码改成对新字体描述代码，重启，系统和应用就开始使用新字体了。到这里，整体的替换过程已经完成，是不是很简单？当然，修改 XML 可能需要对 HTML 有所了解，如果不了解，对照原文件，照猫画虎一样可以完成，没有什么难度。本文貌似已经可以结束了。。

实际上还是有几个值得注意的点，稍不注意可能还是个蛮大的坑。

1. 权限问题

一定要注意权限，否则开机后文字无法正常显示，应用闪退，甚至无法开机。对照原有字体给新字体修改权限：

$ chmod 0744 Lato-Regular.ttf

如果 XML 文件也是新的，也要注意权限。建议将原有 fonts.xml 备个份，拷贝一份新的慢慢修改。

2. 锁屏时间冒号乱码问题

改完英文字体后，发现锁屏时间的冒号变成了一个问号，即无法正常显示，查了下是新字体的问题。因为该字符调用的是码点为 EE01 的字符，而我使用的新的 Lato 里面没有这个字符。

解决的办法是用字体编辑器给相应的字体增加这个码点，并给他相应的字符。可以直接采用其中的冒号，也可以用其他的，这都无所谓的。只需要修改使用到的字体即可，不用都改。我改的是 Lato-Light.ttf。

3. 关于中文字体

Android 默认的中文字体其实也是思源黑体，包含在 NotoSansCJK-Regular.ttc 里面，毕竟是 Google 和 Adobe 合作开发的。但是字重只有一种，也就是说我看到的加粗或黑体的中文都是假的，所以我要增加多重字重。查看 XML 文件，下面这一行注释是明显的分界线，上面部分是英文，下面部分是其他语言。

<!-- fallback fonts -->

在下面部分找到中文部分：

<family lang="zh-Hans">
  <font weight="400" style="normal" index="2">NotoSansCJK-Regular.ttc</font>
</family>

只有一个400的字重，按照原样扩展一下咯：

<family lang="zh-Hans">
  <font weight="300" style="normal">SourceHanSansSC-Light.otf</font>
  <font weight="400" style="normal">SourceHanSansSC-Regular.otf</font>
  <font weight="500" style="normal">SourceHanSansSC-Medium.otf</font>
  <font weight="700" style="normal">SourceHanSansSC-Bold.otf</font>
  <font weight="900" style="normal">SourceHanSansSC-Heavy.otf</font>
</family>

SourceHanSansSC 就是到官方 GitHub 下载的思源黑体简体中文版。总共有7种字重，因为太细的一般不怎么用到，为了节省 Root 空间，我只用了这5种。

发现原来的代码里面有个 index=“2”，字体格式也不太一样。这是因为 NotoSansCJK-Regular.ttc 包含了简、繁、日、韩 四种语言字体，其扩展名表示 TrueType-Collection。需要指示一下是哪种语言。

4. Emoji 也可以换 😃

Emoji 也是一种字体，Android 默认 Emoji 是 NotoColorEmoji.ttf。换 Emoji 比换字体更加方便，因为只有一种。XDA 论坛上可以找到别人制作的 Emoji，包括 IOS、微软等，各种都有。不过换到最后，我还是觉得 Android 默认的 Emoji 更可爱。
