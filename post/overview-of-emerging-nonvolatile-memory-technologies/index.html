新兴非易失性存储技术概述

本文翻译自Jagan Singh Meena等人撰写的“Overview of Emerging Non-volatile Memory Technologies”一文。

不定期更新且水平极其有限，还望海涵。



摘要


   硅基电子设备中的非易失性存储技术可以追溯到20世纪90年代。由于传统闪存面临着物理缩放限制，在当时，铁电场效应晶体管（FeFET）是最有希望取而代之的器件之一。现在，被称为闪存的一系列电荷存储器被广泛应用于手机、音乐播放器等诸多消费电子产品中，而基于NAND闪存的固态磁盘（SSD）正越来越多地取代硬盘驱动器作为笔记本电脑、台式电脑、甚至数据中心中的主存储设备。闪存集成能力逼近极限，许多用来代替传统闪存的新类型存储器也已经被提出来了。相比于普遍的消费电子设备（包括数码相机、手机和便携式音乐播放器等）中成本高昂的硅芯片，新兴存储器技术有望以更低的成本存储更多数据。随着研究的进行，它们将作为现有存储器在未来计算系统中的潜在替代品。 新兴的非易失性存储技术（如磁随机存取存储器（MRAM），自旋转移力矩随机存取存储器（STT-RAM），铁电随机存取存储器（FeRAM），相变存储器（PCM）和电阻随机-存取存储器（RRAM）等）结合了静态随机存取存储器（SRAM）的速度、动态随机存取存储器（DRAM）的密度以及闪存的非易失性。因此，作为未来存储器层次结构的另一种可能性，这项技术变得非常有吸引力。近年来，许多其他新兴的存储技术，如透明塑料、三维（3-D）和量子点存储技术也获得了极大的普及。接下来，毫不夸张地说，计算机存储器的贸易会规模化，产品会变得像塑料仿冒品那样廉价，这个趋势将很快获得最终的商业验证。因此，本综述基于对先进闪存器件最新进展的研究，致力于阐述新型存储技术的快速发展以及科学进程的变迁。

关键词 ： 新兴非易失性存储技术；磁存储器；市场存储技术；忆阻器；相变存储器；随机访问存储器；闪存技术；三维存储器；透明存储器；一致性存储器


结论


   本文针对MOS系列中几种常规类型Flash的存储器架构和规模扩大趋势，从历史发展、最新进展以及未来发展趋势等方面进行了回顾与展望。伴随着新兴存储技术的巨大进步，在对下一代存储器产品进行深入了解之前，我们先讨论了当前的趋势和局限性。在过去的35年中，半导体存储器系列已经极大扩展，取得了更高密度，更快速度，更低功耗，更多功能和更低成本等诸多成就。在过去的40到50年间，非易失性半导体存储器（NVSM）已从浮置栅（FG）技术发展出FAMOS、SAMOS、闪存、多级单元、RRAM、3-D结构和TF-RRAM等技术。自1990年以来，NVSM作为一种启发式技术，引领了数字时代，促进了所有现代电子系统的发展，并为人类带来了前所未有的利益。与此同时，各类存储器中存在的一些限制也逐渐显现。随着器件尺寸减小到十亿纳米范围，NVSM面临许多严重的扩大规模的挑战，例如相邻单元间的接口，存储电荷的减少和随机电信号噪声。因此，我们希望（并有信心）会有一些新兴技术超越这些限制，并有可能取代所有或大部分现有半导体存储技术成为通用半导体存储技术。尽管有这些限制，传统半导体存储器领域仍将继续蓬勃发展，存储器件设备科学家将找到应对这些挑战的方法，甚至可能为未来电子系统开发出低成本、高性能和高可靠性的“统一存储器”。在可行的新型电阻存储技术方面取得的进展依赖于充分了解响应开关和电荷传输的机制、失效机制以及与材料可靠性相关的因素。 此外，当前基于氧化还原的电阻转换技术的发展将有助于改进我们的旧技术。并且进一步的研究将产生更多的令人印象深刻的结果，这将有利于工业和社会改善全世界数十亿人的生活质量。
