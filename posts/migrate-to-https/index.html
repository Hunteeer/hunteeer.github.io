启用 HTTPS

2018-03-31
折腾博客
354
    • 证书申请

Let’s Encrypt 的免费证书，几步便可为博客开启 HTTPS。

Let’s Encrypt 是一个于 2015 年三季度推出的数字证书认证机构，旨在以自动化流程消除手动创建和安装证书的复杂流程，并推广使万维网服务器的加密连接无所不在，为安全网站提供免费的 SSL/TLS 证书。主流浏览器均已支持。

Certbot 是 Let’s Encrypt 官方推荐的获取证书的客户端，自动化申请、处理证书。

证书申请

$ sudo certbot certonly -d "*.stonebythesea.org,stonebythesea.org" --manual --preferred-challenges dns-01 --server https://acme-v02.api.letsencrypt.org/directory

接下去提示输入 Email，以及一些确认选项。最后一步需要在 DNS 解析服务商那儿添加一条 TXT 记录，用来验证 DNS。很快就好了。最后生成证书和密钥：

$ tree /etc/letsencrypt/live/stonebythesea.org/
├── cert.pem
├── chain.pem
├── fullchain.pem
├── privkey.pem
└── README

Let’s encrypt 的免费证书默认有效期为 90 天，到期后如果要续期可以执行：

$ certbot renew

配置 Nginx

配置 HTTPS：

server {
    listen       443 ssl;
    server_name  localhost;

    ssl          on;
    ssl_certificate          "/etc/letsencrypt/live/stonebythesea.org/cert.pem";
    ssl_certificate_key      "/etc/letsencrypt/live/stonebythesea.org/privkey.pem";
    ssl_trusted_certificate  "/etc/letsencrypt/live/stonebythesea.org/chain.pem";

    ssl_session_cache        shared:SSL:1m;
    ssl_session_timeout      5m;

    ssl_ciphers              HIGH:!aNULL:!MD5;
    ssl_prefer_server_ciphers  on;

    location / {
        root   /path/to/root;
        index  index.html index.htm;
    }
}

设置 HTTP 强制跳转 HTTPS：

server {
    listen        80;
    server_name   localhost;
    return        301 https://$host$request_uri;
}
