grep - 文档内部的搜索工具

grep 是一个文档内部的搜索工具。可以根据给定的正则表达式匹配文档内的的文本行。名称来自于g/re/p（Globally search a regular expression and print，以正则表达式进行全域查找以及打印）。

Usage

开始之前，先创建两个测试文件，目录结构及内容如下：

$ cd test; ls
file01 file02
$ cat file01
go
goo
good
google
GOOGLE
$ cat file02
L1 go
L2 goo
L3 good
L4 google
L5 GOOGLE
搜索单个文档
$ grep 'goo' file01
goo
good
google
搜索多个文档
$ grep 'goo' file01 file02	#直接输入多个文档名
file01:goo
file01:good
file01:google
file02:L2 goo
file02:L3 good
file02:L4 google
$ grep 'goo' *	#搜索 test 文件夹内所有文档
file01:goo
file01:good
file01:google
file02:L2 goo
file02:L3 good
file02:L4 google
$ grep 'goo' * --include file02	#指定只搜索 file02
file02:L2 goo
file02:L3 good
file02:L4 google
$ grep 'goo' * --exclude file02	#指定不搜索 file02
file01:goo
file01:good
file01:google
利用管线命令
$ cat file01 | grep 'google'
google
匹配多个 pattern
$ grep -e 'good' -e 'google' file01
good
google

如果要搜索的 pattern 是 ‘-’ 开头的，必须要使用 -e

输出行数、字符偏移
$ grep -n 'goo' file02	#显示匹配到的文本行的行数
2:L2 goo
3:L3 good
4:L4 google
$ grep -b 'goo' file02	#显示匹配到的文本的字符偏移
6:L2 goo
13:L3 good
21:L4 google
输出前后行
$ grep -A 2 'good' file02	#连带输出匹配到的文本行的后2行
L3 good
L4 google
L5 GOOGLE
$ grep -B 2 'good' file02	#连带输出匹配到的文本行的前2行
L1 go
L2 goo
L3 good
$ grep -C 1 'good' file02	#连带输出匹配到的文本行的前后1行，默认是2行
L2 goo
L3 good
L4 google
忽略大小写
$ grep -i 'Google' file01
google
GOOGLE
仅输出匹配次数
$ grep -c 'go' file01
4
仅输出匹配文本
$ grep -o 'go' file01
go
go
go
go
输出未匹配的文本行
$ grep -v 'go' file01
GOOGLE
静默匹配
$ grep -q 'google' file01	#无论是否匹配到，无任何标准输出。匹配到返回0
$ grep -q 'great' file01	#未匹配到返回1
